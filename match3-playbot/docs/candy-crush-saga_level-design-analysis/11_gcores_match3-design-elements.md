## 引言

随着未成年人游戏限制、版号慢发、短视频冲击、经济发展和游戏同质化等因素，游戏轻量化趋势愈发明显。例如：
- 《崩坏：星穹铁道》五分钟做完日常任务
- 《出发吧麦芬》《咸鱼之王》《一念逍遥》挂机变强
- 《羊了个羊》每日一关

游戏对玩家时间的要求越来越低。

尽管游戏行业整体下滑（参考《2024 年移动游戏市场报告》），但休闲向游戏逆势上扬。**Royal Match** 在发行两年后随着新道具上线，流水暴增 1.5 倍，拿下全球手游收入榜第三名（2024 年 5 月数据）。《蛋仔派对》也公布了三消新作《蛋仔滑滑》，莉莉丝布局三消已两年。三消变成了数据优秀、被大厂看好的品类。

作者将三消分为三个大模块：**1. 元素，2. 关卡，3. 外围系统**。本篇聚焦"元素"。

---

## 1. 元素

### 1.1 元素手感

元素手感可类比动作游戏的"打击感"，分解为三部分：
- **前摇** → 滑动
- **释放** → 消除
- **后摇** → 掉落

这是最基础的手感，需要长期打磨，立项时就要逐帧分析头部产品的消除动画。

**各产品对比**：

| 产品 | 元素设计 | 消除/掉落手感 |
|------|---------|--------------|
| Project Makeover (PM) | 针脚围绕的坐垫/抱枕，符合更衣/美容世界观 | 气泡感消失 + 光点，掉落晃动 Q 弹 |
| 梦幻家园 | 家中常见物件、管家奥斯丁的蝴蝶结 | 直接坍缩消失，元素晃动如纸片，舒适 |
| 糖果传奇 | 硬糖质感的糖果 | 原地生成气球式破碎，元素晃动坚硬 |

总体比较下，**柔软的质感似乎更适合三消**。虽然目前没有足够数据证明柔软更舒适，但从程序与美术层面来说，"柔软"效果需要更多素材和代码，能与制作质量不高的产品形成区隔。

**蛋仔滑滑**（最近开测）：虽然是蛋仔 IP，但元素与蛋仔不够贴合，消除时是破碎效果（与 Royal Match 如出一辙），但蛋仔本应是 QQ 弹弹的。底部"关内道具" icon 强行融合蛋仔 IP，且把蛋仔放在后面占据大部分空间，玩家难以辨识道具效果，可能造成误操作。作者认为目前手感尚不足以跻身第一梯队。

**崩坏：星穹铁道**：更新了三消活动玩法，元素利用光噪点和去阴影打造 2D 效果，元素圆润；玩法上增加了不同技能效果，关卡目的简单（尽可能多消除），随机性与策略性强，地图设计精致、节奏好。

### 1.2 事件

事件 = 道具生成 + 道具效果 + 道具组合。

#### 1.2.1 道具生成类型

道具类型仅限：
- 【点消】→ 飞机
- 【横竖消】→ 火箭
- 【范围消】→ 炸弹
- 【单色消】→ 彩球

道具触发方式（单击/双击）也会影响手感。

**Royal Match 新出【双色消】**：在游戏内给人 2 倍消除效果，玩家获得后难以割舍。这个道具直接让收入抬高了 1.5 倍，目前还未被同行广泛沿用。

#### 1.2.2 道具效果

道具是否相互影响决定了游戏难度基调：
- 若【火箭】【炸弹】可以触发【彩球】，彩球大概率被白白浪费
- 若不能触发，玩家可以积攒道具与彩球形成组合，造成强力消除

这必须在关卡设计最初期决定好，后期难以改变。

**不同时期道具的作用**：
- 前期：【火箭】帮助打开地图障碍；【飞机】影响甚微（尤其多层障碍关卡）
- 后期：【飞机】精准打击作用凸显
- 进入关卡时是否携带【横竖消】【范围消】，将很大程度影响开图所需步数
- 关卡末期：可操作区间、元素种类数量影响【飞机】合成

**Royal Match 的【飞机】智能程度极高**：每次选取的目标都是让玩家收益最大化的目标，提升体验、避免弃游。这需要：
- 策划对障碍优先级做出明确排序
- 程序能计算当前关卡内的最高收益格子

#### 1.2.3 道具组合

四种道具任意结合的效果远高于单独触发。

高难度关卡中，玩家是否在连胜中获得了大量"连胜道具"带入关卡，将很大程度决定首次过关概率。一次组合可能带来 **超过 10 步的收益**。当玩家步数为 0 时只剩寥寥几个目标未达成，付费欲望就会高涨。

**Supercell 新作《爆裂小队》**（最近爆火，下载榜第一）也有类似设计：
- 限时双倍宝箱奖励
- 连胜中断可用金币保连胜
- 战斗时可用道具开宝箱

这些与三消的付费设计有重合，只是把单机付费设计应用到了 PVP 中。

**玩家心理三段式**：
- a. 携带了多个道具入场，输了重来不知道还能不能过
- b. 这局输了，之前连胜的奖励就没了，过关概率更小
- c. 只要买 5 步我就可能过关，既保住连胜奖励又获得过关奖励

在"失去的恐惧"与"获得的喜悦"双重诱惑下，玩家付费意愿达到高峰。

**Callback**：【双彩球】的出现，加大了玩家潜在会失去的奖励（即连胜的沉没成本），让玩家更不想失去，付费意愿达到更高峰，促使了 Royal Match 收入上涨。

## 本章小结（原文）

元素在消除游戏中的手感和事件（道具生成及效果）对玩家体验和游戏商业化有重大影响。
- **手感**：涉及滑动、消除、掉落三个阶段，元素设计应与世界观相符
- **道具生成**：包括点消、横竖消、范围消和单色消，Royal Match 新增双色消显著提高收入
- **道具效果**：相互影响决定了游戏难度和玩家策略，组合效果可带来显著收益
- **高难度关卡**：携带道具提高过关概率，玩家在失去连胜奖励与过关奖励的双重诱惑下，付费意愿增加
- **Royal Match** 的高智能飞机设计提升了玩家体验，增加了游戏黏性和收入
